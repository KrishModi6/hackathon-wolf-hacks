No Smartphone? Use SMS Triage

This equity-first access enables residents without data plans or smartphones to get triage recommendations via text.

Try SMS Triage

Example SMS Flow
To: 647-TRIAGE-1
User: TRIAGE L6S AGE 67 SYMPTOMS fever confusion
SwiftCare:
⚠️ URGENT PATTERN: Sepsis risk
→ Peel Memorial ER (wait: 45 min)
Reply QR for intake code
User: QR
SwiftCare:
Handoff: TW-94KQ2M
Show to intake nurse
⚠️ Call 911 if breathing worsens

Why SMS Triage?

  • Universal access: Works on basic phones with no data plan.
  • Language support: SMS can be automatically translated to French, Mandarin, Punjabi, etc.
  • Community reach: Accessible to seniors, newcomers, and vulnerable populations.
  • Same AI: Uses the same CTAS + ML early detection as the web version.
  • QR handoff: Generates printable intake codes for paper-based workflows.
